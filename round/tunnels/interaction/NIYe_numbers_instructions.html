Numbers Code Instructions

The puzzle consists of a 100-digit number, which is “encoded” in the following way: the digit in the Mth position of the encoded string is the Nth digit of the unencoded string, where N is the length of the Collatz sequence that takes M to 1.

The Collatz conjecture states that for any positive integer, we can generate a sequence that terminates in 1 via the following rules: if the integer is even, divide it by two. If the integer is odd, multiply by 3 and add 1.

So, for example, for the first digit in the puzzle we have the Collatz sequence:

1

This is a sequence of length 1, so the first digit of the encoded string is also the first digit of the decoded string.

For the third digit, we have the Collatz sequence:

3, 10, 5, 16, 8, 4, 2, 1

This is a sequence of length 8, so the third digit of the encoded string is the eight digit of the decoded string.

Decoding it in this fashion gives the new, shorter string [redacted], which is padded indefinitely with zeroes at the end. Interpreting this using A1Z26 gives the answer, [redacted].
