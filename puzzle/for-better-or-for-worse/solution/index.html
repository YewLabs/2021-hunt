The puzzle is presented as a set of 40 images, followed by a 10×10 grid. In the set of 40 images, the characters depicted are generally unhappy.

The characters are unhappy because they are using the wrong tools for the job. Some of them are using tools that are too poor, and need to be made better, and some are using tools that are too good, and need to be made worse.

For example, this person is using butt to flavor their bread. They need better butt, or butter, to be satisfied.

Meanwhile, the officer of the law is attempting to verify her credentials using a badger. The badger is too badge for the task, and needs to be worse; that is, a badge would work.
