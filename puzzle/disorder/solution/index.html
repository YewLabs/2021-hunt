Googling these seven countries yields various results on strategy websites, forums and wikis for the board game Diplomacy. Reading a little bit about Diplomacy, we might learn that there are 34 supply centers, corresponding to our 34 clues, and that each territory in a Diplomacy game is represented by a three-letter code; both of these help us be sure that Diplomacy is the next step.

Knowing that each territory in Diplomacy is represented by a three-letter code, we now look for those three-letter codes among the trigrams we were given. We find that each clue has two or three of them.

At this point it behooves us to learn more about the rules of Diplomacy. In Diplomacy, each turn culminates with all of the seven players writing down their orders, usually abbreviated using the three-letter codes, for each of their units. Another explanation of orders can be found here; in short, each order either moves a unit from one territory to another, keeps a unit in the same place, or it interacts with another order that is one of the first two types. The title, combined with the two to three relevant trigrams in each clue, suggests that orders are relevant. Using the proper order of the clue, we can make a consistent set of orders from the trigrams in each clue. The country associated with the answer from the clue is assumed to have given the order.

These orders are enough to reconstruct most of the state of the game; there are 34 orders, so we know exactly how many units each country has because there can never be more units than supply centers on the board. We can construct the outcome of the turn represented by these orders, either by manually arranging units on a Diplomacy board, or by using one of several tools (Backstabbr, use Sandbox Mode, jDip).

Playing out these orders, we can see which ones succeeded and which ones failed.

Of particular interest to us are the seven places where one country has taken a territory from another. All of them are different, and all of them are supply centers.

Noticing that there are seven displacements, we look closer and notice that each country has displaced one other, and they make a cycle:
