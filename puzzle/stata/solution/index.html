When we solve a puzzle, some number of blanks appear on the meta page to the right of the answer. The above message suggests that we should change each feeder answer somehow to produce a new word that matches the enumeration given by the blanks.

Eventually we realize that we can “fold” each answer to produce a new object. For example we can fold a COMPUTER to get a LAPTOP and fold a PIZZA to get a CALZONE.
