This is a Rows Garden puzzle with a twist. The first immediate clue that something is off is that there is one fewer row than there ought to be. If we count the "Around" clues and the "More Around" clues, we find that there are ten and fourteen respectively. There are ten white blooms and fourteen each of gray and pink blooms, so not all the blooms can be clued.

Once we make some progress on filling in the words, we may notice that some of the rows seem too short. Moving things around, we eventually must realize that if we place exactly one letter in each pink bloom, the spacing works out. This also lines up nicely with the number of clues we have of each color, and explains the missing top row.
