Infinite Corridor Simulator is the fifth and final type of puzzle unlocked in the Infinite Corridor round. True to its name, each Infinite Corridor Simulator puzzle looks a lot like a summary of the Infinite Corridor Round itself — more specifically, each Infinite Corridor Simulator puzzle contains a list of 100,000 similarly named and numbered puzzles along with their answers. These answer/puzzle pairings differ between different instances of Infinite Corridor Simulator and do not match the puzzles and answers in the Infinite Corridor round (with one notable exception). However, all Infinite Corridor Simulator puzzles use the same puzzle mechanics as each other.

If we take the first letters of each answer, we get the message DECODE EACH PUZZLE (after which the first letters devolve into nonsense). This suggests looking at all the answers of each type separately. When we do this, we find that we can solve each stream of answers as a subpuzzle to get an answer phrase of the form “[ordinal number] [digit]” (e.g. FIRST SEVEN). The solutions to each subpuzzle are detailed below.

The five ordinal numbers we obtain always include each of FIRST, SECOND, THIRD, FOURTH, and FIFTH exactly once. For example, in Puzzle 5, we obtain the five subpuzzle answers:

FIRST ONE
SECOND SEVEN
THIRD ZERO
FOURTH THREE
FIFTH ONE

This allows us to form a five-digit number n by placing each digit at its corresponding location (17031 for the above example). Looking up Puzzle n in our list of puzzles, we find it is an Infinite Corridor Simulator puzzle — for example, in Puzzle 5, we observe:

...
Puzzle 17030Library of ImagesDUN
Puzzle 17031Infinite Corridor SimulatorDRESS
Puzzle 17032Infinite Corridor SimulatorEXILE
...

The answer to Puzzle n is the answer to this Infinite Corridor Simulator puzzle (and so for the specific example of Puzzle 5, the answer is DRESS).

All of these answers have even length. In addition, the central bigrams of these answers often repeat (IR, ER, HI, ER, TH, RO, TH, ...). These central bigrams are randomly selected from the answer phrase — in particular, they alternate between being chosen from the first word of the answer phrase and being chosen from the second word of the answer phrase. From this, we can reconstruct the answer to this subpuzzle. For this specific puzzle (Puzzle 5), this leads to the subpuzzle answer THIRD ZERO.

If we look at the lengths of these answers, we see that in order they are 3, 1, 4, 1, 5, 9, 2, 6, …, which we can recognize as the digits of pi. Indeed, the lengths of the answers match up with the digits of pi, with the exception that where the corresponding digit would be 0, the answer is a single character. The first such instance occurs at the 33rd digit of pi (counting the leading 3 as the first digit) — in the above example, this corresponds to the 33rd instance of Library of Images, which is Puzzle 153 with the answer F. Taking these characters in order reveals the answer to this subpuzzle. For this specific puzzle (Puzzle 5), this leads to the subpuzzle answer FOURTH THREE.

Many of these answers start with a compass direction (N, S, E, W) and some (e.g. SEANCE) even start with one of the four diagonal compass directions (NE, NW, SE, SW — however, NW will not appear in any puzzle). These answers are divided into chunks by answers that do not start with any compass direction (e.g. FLANK). The path we obtain by following the compass directions for a single chunk draws out a letter. If we read the first letters of these answers, we find that this almost matches the string we get by concatenating all the answers in order. For example, the string we get by reading the first letters of the above answers is

DUKYUDDRKETHYUGU…

and the string we get by simply concatenating the answers is

DUSKYUDDERKETCHYOUNGU…

It is possible to transform the second string into the first string by deleting some letters (approximately one per word, but occasionally more, and occasionally fewer). Reading the deleted letters, we find the subpuzzle answer. For this specific puzzle, the answer is SECOND SEVEN.
