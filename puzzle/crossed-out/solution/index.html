In each image, something has been crossed out.

Row one: XX (roman numeral for twenty), Jamie Foxx, Sexual XXXXX!
Row two: XX (chromosomes), XXXX (from Discworld)
Row three: XXXXX (acromantula danger rating)
Row four: T.J. Maxx, XXXXXXXXXXXX (twelve strikes in bowling)
Row five: Dos Equis (XX), Exxon, Rexxar, the flag of Amsterdam (XXX)

Clearly, the number of consecutive Xs in each picture is relevant. But what do we do with them? Well, the puzzle is about Xs, and X means multiply, so we multiply them together.

2 x 2 x 5 = 20 = T
2 x 4 = 8 = H
5 = 5 = E
2 x 12 = 24 = X
2 x 2 x 2 x 3 = 24 = X

So the answer to the puzzle is, fittingly enough, THE XX.
